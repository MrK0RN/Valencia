## Создание демо-администратора

### Способ 1: Через PHP-скрипт (рекомендуется)

```bash
php setup_demo_admin.php
```

Этот скрипт создаст или обновит демо-администратора в базе данных.

### Способ 2: Через SQL

Демо-администратор автоматически создается при инициализации базы данных через `init.sql`.

Если нужно создать вручную:

```sql
INSERT INTO users (name, email, password, role, notification_enabled)
VALUES (
    'Демо Администратор',
    'admin@demo.com',
    '$2y$10$92IXUNpkjO0rOQ5byMi.Ye4oKoEa3Ro9llC/.og/at2.uheWG/igi',
    'admin',
    true
);
```

**Важно:** Пароль в SQL уже захеширован. Для создания нового хеша используйте PHP-скрипт.

## Безопасность

⚠️ **ВНИМАНИЕ:** Демо-доступ предназначен только для разработки и тестирования!

В продакшене:
1. Измените пароль демо-администратора
2. Или удалите демо-пользователя и создайте реального администратора
3. Используйте сильные пароли

## Функционал админ-панели

- ✅ Управление объектами недвижимости (CRUD)
- ✅ Drag-and-drop сортировка объектов
- ✅ Управление пользователями
- ✅ Просмотр статистики
- ✅ Управление избранным и историей просмотров
- ✅ Система уведомлений

## Личный кабинет покупателя

После входа как обычный пользователь доступен личный кабинет:
- `/account/index.php` - главная страница
- `/account/favorites.php` - избранное
- `/account/history.php` - история просмотров
- `/account/settings.php` - настройки уведомлений
